Smart Resume Screening & Interview Preparation Assistant
Humane, LLM-Powered Decision-Support System for Recruiters

📌 Overview

The Smart Resume Screening & Interview Preparation Assistant is a modular, LLM-driven web application designed to support recruiters in making transparent, explainable, and humane hiring decisions.

Unlike traditional Applicant Tracking Systems (ATS) that rely on keyword filtering and automated rejection, this system:

* Uses a Large Language Model (LLM) for structured resume parsing and semantic evaluation

* Provides category-based scoring with clear explanations

* Generates tailored interview questions

* Preserves full human-in-the-loop control

* Does not automatically reject or hide candidates

This project demonstrates modern AI system design, ethical AI principles, and modular software architecture.
